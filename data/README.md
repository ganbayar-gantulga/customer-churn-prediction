# Өгөгдлийн тайлбар

## Telco Customer Churn Dataset

**Эх сурвалж:** [Kaggle - Telco Customer Churn](https://www.kaggle.com/datasets/blastchar/telco-customer-churn)

### Татах заавар:
1. Kaggle дээр бүртгүүлнэ
2. Дээрх линкээр орж "Download" дарна
3. `WA_Fn-UseC_-Telco-Customer-Churn.csv` файлыг `data/raw/telco_churn.csv` нэрээр хадгална

### Баганууд:
- **customerID** — Харилцагчийн ID
- **gender** — Хүйс
- **SeniorCitizen** — Ахмад эсэх (0/1)
- **Partner** — Хамтрагчтай эсэх
- **Dependents** — Хамааралтай хүнтэй эсэх
- **tenure** — Хэдэн сар үйлчлүүлсэн
- **PhoneService** — Утасны үйлчилгээ
- **MultipleLines** — Олон шугам
- **InternetService** — Интернэтийн төрөл
- **OnlineSecurity** — Онлайн аюулгүй байдал
- **OnlineBackup** — Онлайн нөөцлөлт
- **DeviceProtection** — Төхөөрөмж хамгаалалт
- **TechSupport** — Техникийн дэмжлэг
- **StreamingTV** — ТВ стриминг
- **StreamingMovies** — Кино стриминг
- **Contract** — Гэрээний төрөл
- **PaperlessBilling** — Цаасгүй төлбөр
- **PaymentMethod** — Төлбөрийн арга
- **MonthlyCharges** — Сарын төлбөр
- **TotalCharges** — Нийт төлбөр
- **Churn** — Гарсан эсэх (Yes/No) ← Зорилтот хувьсагч
